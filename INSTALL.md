# Installing News To Me

Follow these steps to install News To Me on your Windows computer.

## Requirements

- Windows 10 or later
- Administrator privileges for installation (Windows will prompt you)
- Approximately 150-200 MB of free disk space

## Installation Steps

1. **Download the Installer**
   - Download the `News To Me Setup.exe` file from the download link

2. **Run the Installer**
   - Double-click the downloaded `.exe` file
   - Windows may show a security warning. Click "Run anyway" or "Yes" to proceed
   - The installer wizard will open

3. **Choose Installation Directory**
   - The installer will suggest `C:\Program Files\News To Me\`
   - Click "Browse" if you want to change the location
   - Most users should use the default location
   - Click "Next" to continue

4. **Create Shortcuts** (Optional)
   - Check "Create Desktop Shortcut" if you want an icon on your desktop
   - Check "Create Start Menu Shortcut" (usually checked by default)
   - Click "Install" to proceed

5. **Installation Complete**
   - The installer will extract files and set up the application
   - When complete, you'll see a "Finish" button
   - Check "Run News To Me now" if you want to launch it immediately
   - Click "Finish"

## Launching News To Me

After installation, you can launch News To Me in three ways:

1. **Desktop Icon**: Double-click the "News To Me" icon on your desktop

2. **Start Menu**: 
   - Click the Windows Start button
   - Search for "News To Me"
   - Click the application to open it

3. **Program Files**:
   - Open the News To Me folder at `C:\Program Files\News To Me\`
   - Double-click `News To Me.exe`

## First Launch

When you launch News To Me for the first time:

1. The backend server will start automatically
2. Your default web browser will open automatically
3. You'll see the News To Me interface
4. If the browser doesn't open automatically, use the URL shown in the terminal window

## Uninstalling News To Me

To uninstall News To Me:

1. Open **Control Panel**
2. Go to **Programs** → **Programs and Features**
3. Find "News To Me" in the list
4. Click it and select **Uninstall**
5. Confirm the uninstall when prompted
6. Windows will remove all application files

Alternatively:
- Right-click the Start Menu shortcut and select "Uninstall"
- Or run the uninstaller from `C:\Program Files\News To Me\Uninstall News To Me.exe`

## Troubleshooting

### Application Won't Launch

**Issue**: Browser doesn't open automatically
- **Solution**: 
  - Open your web browser manually
  - Navigate to `http://localhost:3001`
  - You should see the News To Me interface

**Issue**: Port 3001 already in use
- **Solution**: 
  - Close other applications using port 3001
  - Restart the News To Me application

**Issue**: "Permission Denied" error
- **Solution**:
  - Run News To Me with administrator privileges
  - Right-click the News To Me icon and select "Run as Administrator"

### Installation Issues

**Issue**: "Cannot find the application" during installation
- **Solution**:
  - Right-click the installer and select "Run as Administrator"
  - Disable antivirus software temporarily during installation
  - Try installing to a different directory

**Issue**: Installation fails with error
- **Solution**:
  - Ensure you have at least 200 MB free disk space
  - Close other running applications
  - Restart your computer and try again

### Performance Issues

**Issue**: Application is slow to start
- **Solution**:
  - First launch may be slow as the backend initializes
  - Subsequent launches will be faster
  - Ensure your computer meets the requirements

## Getting Help

If you encounter issues:

1. Check that your internet connection is working
2. Try restarting the application
3. Try restarting your computer
4. Uninstall and reinstall the application

For additional support, contact the News To Me team.
